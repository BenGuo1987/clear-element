# Clear Element
Clear Element 是一个谷歌扩展程序它可以让你移除那些你不想看到的网页内容。

[官方文档](https://developer.chrome.com/extensions)

## 1. 什么是扩展程序（Google Extensions）
谷歌扩展程序是定制浏览体验的小型软件程序。 它们使用户能够根据个人需求或偏好定制Chrome功能和行为。 它们基于HTML，JavaScript和CSS等Web技术构建。

  一些常用的谷歌插件
  - **Adblock Plus**
  - **Infinity 新标签页(Pro)**
  - **Stylus**
  - **better-onetab**
  - **Google翻译**
